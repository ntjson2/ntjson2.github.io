MineralFi

MineralFi

  • Category: Web3 Geospatial DeFi Platform
  • Type: Smart Contract + API Infrastructure
  • Stack: Solidity, Next.js, GraphQL, DynamoDB, IPFS
  • Timeline: Jul 2023 – Present
  • Role: Lead Blockchain & Cloud Engineer

What it does

MineralFi lets you tokenize real-world assets—mineral rights, conservation land, geospatial investments. I built the technical foundation: contracts, APIs, and infrastructure. The platform is designed so real humans can manage mineral assets as tokens, and down the road, autonomous agents could run strategies on it.

What I shipped

  • Smart contracts: MINFI token (ERC-20) and NFTs (ERC-1155) that you can actually upgrade without blowing everything up. Handles high-value transactions safely.
  • Reward mechanisms: NFTs that pay out when milestones hit—not just theoretical future stuff, actually working now.
  • Uniswap plumbing: Hooked into V3 for liquidity and yield. The math checks out and it improved capital efficiency about 30%.
  • Real-time dashboards: Next.js + GraphQL so investors can watch their portfolio in live time. Data structured so it's clean for queries.
  • Geospatial visuals: 3D maps with ThreeJS. Data lives on IPFS so you can verify what you're looking at is real.
  • DevOps that doesn't break: Proper CI/CD with GitHub Actions, contract testing with Hardhat, and actual staging before production. Cut incidents way down.

Where it goes

Built the APIs and data formats so if we ever wanted to run autonomous agent strategies on top of it, the door's open. Right now it's humans managing mineral assets via a clean interface. But the architecture is ready for the next step.
