Project details

FormPro is a long running software suite that combines a front-end, customizable client application with and back-end data management service. This software serves as a repository and interactive service for an authoritative collection of evidence-based data for long-term patient care needs. This software primarily targets life care planners, legal nurse consultants, paralegals, plaintiff and defense attorneys.

On this project I learned to be organized and professional as many of our clients and software expectations were higher than normal.
