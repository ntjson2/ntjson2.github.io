Yay Baby

  • Category: Unity based mobile app
  • Client: Android users
  • Project finish date: May, 2023
  • Project URL: amarustudios.com

Project details

Yay Baby was started in 2017 and has been an ongoing Android app project aimed at assisting families needing potty training help via positive reinforcement.
This project uses Unity's built-in cross-platform technology to deliver this software to Android, iOS and eventually WebGL based systems.

Yay Baby has been a learning experience on the front of project estimates. I originally estimated this project to take a week at most to have the first sprint done. It has been four years! This is mostly due to feature creep, loss of co-worker and challenges and inefficiencies around relearning a code base after leaving and coming back to the project repeatedly.
