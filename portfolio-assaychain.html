Portfolio Details

AssayChain

  • Category: Full Stack Web3 API Service
  • Type: SaaS / Blockchain
  • Stack: Node.js, React, x402 protocol, IPFS
  • Project URL: assaychain.com

Project details

AssayChain delivers structured mineral intelligence through a REST API providing USGS commodity data, field studies, and mineral processing research. The platform combines advanced mineralogy processing methods with on-chain provenance verification. I built this to enable transparent, auditable mineral data delivery - in which AI system are in great need of.

Core Features

  • Processing Methods: Gravity separation, ultrasound cavitation, placer vs. hardrock analysis, recovery rate optimization, and elutriation
  • Data Infrastructure: 14 API routes across 3 authentication tiers delivering real-time USGS commodity summaries, mine production data, and market pricing
  • On-Chain Provenance: MineFlowTrack field data capture, IPFS distributed storage with content addressing, and EAS attestation for immutable verification
  • Scalability: API pricing at $0.10 per call enables machine-readable USGS data access for AI agents and enterprise systems

Technical Implementation

Built full-stack: React frontend for data visualization, Node.js backend for API aggregation and processing, Solidity smart contracts for provenance attestation, and IPFS integration for distributed, tamper-proof data storage. The architecture enables AI agents to query mineral data programmatically while maintaining complete on-chain audit trails.
